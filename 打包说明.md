# AI模拟箱 - 打包说明

## 安装依赖

```bash
cd c:/_dx/javaweb/ch01/src/main/webapp/ai_sandbox
npm install
```

## 开发模式运行

```bash
npm start
```

## 打包成 EXE

```bash
npm run dist
```

## 打包结果

打包成功后，EXE 文件位于：

```
dist-win\AI模拟箱-win32-x64\AI模拟箱.exe
```

这是一个**便携版应用**，整个文件夹（约 150-200MB）可以复制到任何地方运行。

## 包含的应用

- ✅ **狼人杀** - AI 对战的狼人杀游戏
- ✅ **文字冒险** - 文字类文字游戏
- ✅ **对战系统** - AI 对战平台
- ✅ **聊天室** - AI 聊天应用
- ✅ **占卜** - AI 占卜功能
- ✅ **分析器** - 文本分析工具

## Ollama 集成

应用会自动检测并启动 Ollama 服务（如果已安装）。支持的安装位置：

- `C:\Users\{用户名}\scoop\shims\ollama.exe`
- `C:\Program Files\Ollama\ollama.exe`
- `C:\Program Files (x86)\Ollama\ollama.exe`

如果 Ollama 未安装，应用仍可运行，但需要手动配置 API 地址。

## 文件结构

```
ai_sandbox/
├── electron/
│   ├── main.js      # Electron 主进程（含 Ollama 自动启动）
│   └── preload.js   # 预加载脚本
├── analyzer/        # 分析器
├── battle/         # 对战系统
├── chat/           # 聊天室
├── divination/      # 占卜
├── textgame/       # 文字冒险
├── werewolf/       # 狼人杀
├── index.html       # 主页面
├── shared.js        # 共享脚本
├── package.json     # 项目配置
└── dist-win/       # 打包输出目录
    └── AI模拟箱-win32-x64/
        ├── AI模拟箱.exe  # 主程序
        └── ...           # 运行所需文件
```

## 自定义图标

将 `icon.ico` 或 `icon.png` 放在 `ai_sandbox` 根目录，修改 package.json：

```json
"dist": "electron-packager . \"AI模拟箱\" --platform=win32 --arch=x64 --out=dist-win --overwrite --icon=icon.ico"
```

## 注意事项

- 打包后的应用大小约 150-200MB（包含 Electron 运行时）
- 首次运行可能被 Windows Defender 误报，需要添加信任
- 便携版无需安装，直接运行即可
- Ollama 自动启动需要系统管理员权限（首次运行可能需要授权）
