# AI回答分析工具

一个基于Web的智能内容分析工具，使用AI模型检测回答或其他文本内容中的可疑陈述、事实错误、偏见和不准确信息，并构建可查询的公共案例库。

## 🚀 功能特点

- **智能内容分析**：使用先进的AI模型分析文本内容，检测可疑陈述、事实错误、偏见等
- **多种分析模式**：
  - 常规分析：全面检测各类问题
  - 事实核查：专注于事实准确性验证
  - 偏见检测：识别潜在的偏见和倾向性表述
  - 综合分析：多维度全面评估
- **可配置严格程度**：提供宽松、中等、严格三种分析级别
- **案例库系统**：保存分析结果到本地案例库，支持高效搜索和过滤
- **批量分析功能**：支持多案例批量分析和AI总结
- **多格式导出**：支持JSON、Word文档和Excel表格格式导出
- **数据管理**：支持导出、导入和备份所有数据

## 📋 系统要求

- 现代浏览器（Chrome、Firefox、Safari、Edge等）
- 稳定的互联网连接（用于API调用）
- 有效的API密钥（支持自定义API端点）

## 🛠️ 技术实现

- **前端**：纯HTML、CSS、JavaScript实现，无需后端服务器
- **UI框架**：使用现代CSS和响应式设计，适配桌面和移动设备
- **API集成**：支持多种AI模型，包括GLM-4.5、Qwen3系列等
- **数据存储**：使用浏览器本地存储保存案例和设置
- **文档导出**：集成docx、xlsx库支持多格式文档导出

## 🚀 快速开始

1. 在浏览器中打开 `index.html` 文件
2. 进入"设置"标签页，配置您的API密钥和端点
3. 返回"分析回答"标签页，输入待分析的内容和可选的上下文
4. 选择分析类型和严格程度，点击"开始分析"
5. 查看分析结果，可选择保存到案例库
6. 在"案例库"标签页中搜索和浏览已保存的案例
7. 使用"总结分析"功能对多个案例进行批量AI分析

## 📖 使用说明

### 分析内容

1. **上下文（可选）**：提供问题的背景信息，有助于更准确的分析
2. **内容**：粘贴需要分析的文本内容（不仅限于回答）
3. **分析类型**：
   - 常规分析：检测各类问题
   - 事实核查：专注于事实准确性验证
   - 偏见检测：识别潜在的偏见和倾向性表述
   - 综合分析：多维度全面评估
4. **严格程度**：
   - 宽松：仅标记明显的错误和问题
   - 中等：标记可能的错误、不明确表述和潜在问题
   - 严格：详细分析所有可能的细微问题和不准确之处

### 案例库管理

- **搜索**：通过关键词搜索案例内容、标签或分析结果
- **过滤**：按可疑度范围、时间范围和标签筛选案例
- **查看详情**：点击查看完整案例信息，包括上下文、内容和详细分析
- **批量操作**：支持多选案例进行批量导出、分析和删除
- **导出/导入**：支持单个案例或整个案例库的导出和导入

### 总结分析

- **批量分析**：选择多个案例进行AI总结分析
- **趋势识别**：识别案例中的共同模式和趋势
- **问题统计**：统计和总结最常见的问题类型
- **改进建议**：针对发现的问题提供具体的改进建议
- **导出报告**：支持将总结分析导出为多种格式

### 设置配置

- **API密钥**：配置您的API密钥（仅保存在本地浏览器中）
- **分析模型**：选择用于分析内容的模型
  - GLM-4.5（推荐）
  - Qwen3-30B-A3B
  - Qwen3-VL-32B-Thinking
  - QwQ-32B
- **自定义API端点**：配置自定义的API端点地址
- **存储设置**：配置案例数据的存储位置和最大条目数

## 🔧 API配置

工具支持自定义API端点，默认使用 `https://api.suanli.cn/v1`。您可以在设置页面修改API端点以使用不同的AI服务提供商。

支持的模型：
- `zai-org/GLM-4.5`（推荐）
- `free:Qwen3-30B-A3B`
- `Qwen/Qwen3-VL-32B-Thinking`
- `Qwen/QwQ-32B`

## 🔒 隐私和安全

- 所有数据均保存在本地浏览器中，不会上传到任何服务器
- API密钥仅保存在本地，用于直接调用AI API
- 不会收集或传输任何个人数据
- 支持完全离线使用（除API调用外）
